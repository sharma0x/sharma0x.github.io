Caption Generator

Karaoke-style highlighted captions for audio

View on GitHub

01. Overview

Generate karaoke-style highlighted captions for any audio — powered by Manim and OpenAI Whisper. Upload an audio file, get back a video with word-by-word highlighted subtitles synced to the speech.

02. Features

  • ▹ Word-level highlighting — each word lights up as it's spoken
  • ▹ Automatic transcription — uses OpenAI Whisper for accurate speech-to-text
  • ▹ Editable captions — edit the transcript and timestamps auto-align
  • ▹ Beautiful rendering — Manim produces crisp 1080p animations with rounded backdrop strips
  • ▹ Two interfaces — CLI for automation, Streamlit web app for interactive use
  • ▹ Fully configurable — font, colours, highlight style, positioning, and more

03. Tech Stack

Python 3.10+ Manim OpenAI Whisper FastAPI Streamlit FFmpeg

04. Architecture 

Audio File → transcriber (Whisper) → text_alignment → manim_renderer → video_processor → Final MP4

transcriber.py

Whisper transcription → word-level SRT

text_alignment.py

Diff-based timestamp reconciliation for edits

manim_renderer.py

Manim scene with chunk display + word highlighting

video_processor.py

FFmpeg audio track replacement

05. Configuration

Setting Default Description
font_name Arial Black Font family
font_size 72 Text size
text_color #FFFFFF Caption text colour
highlight_color #FFA500 Active word highlight
bg_color #000000 Background strip colour

06. Usage

CLI

Generate a captioned video from an audio file in one command:

python cli.py your_audio.mp3

Streamlit Web App

For an interactive experience with live configuration:

streamlit run app.py
